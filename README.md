# Data Science Final Projects (DS_Final_Projects)

В данном репозитории представлены сборные и выпускной проекты по Data Science, выполненные в ходе практики.

Проекты охватывают полный цикл работы с данными: от анализа и подготовки данных до построения и оценки моделей машинного обучения.

## Проекты

- **final_project_1**  
  Сборный проект, включающий анализ данных, построение моделей и интерпретацию результатов.

- **final_project_2**  
  Сборный проект, направленный на решение прикладной задачи с использованием методов машинного обучения.

- **graduation_project**  
  Выпускной проект, объединяющий все полученные знания и навыки.

## Назначение репозитория

Репозиторий является частью портфолио и демонстрирует практический опыт решения задач анализа данных и машинного обучения.
